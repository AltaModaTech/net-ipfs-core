Interface IDagApi

Manages the IPLD Directed Acrylic Graph.

Namespace: Ipfs.CoreApi
Assembly: Ipfs.Core.dll
Syntax
public interface IDagApi
Remarks

The dag API is a replacement of the IObjectApi, which only supported 'dag-pb'. This API supports other IPLD formats, such as dag-cbor, ethereum-block, git, ...
